# lazeLoadImg
图片懒加载（图片出现在可视区域再加载）

#### 使用方法：
- 引入相应的js文件  
```<script src="js/lazeload.js"></script>```

- 初始化
```
var x = new lazyLoad();
x.init();
```

> **注意：**

```<img src="img/load.jpg" alt="img" lazy-src="img/goods001.jpg" height="640px">```

**src 属性放预加载的图片，lazy－src放实际展示的图片**
